# Pr-diction-de-l-Attrition-Client
Ce projet vise à analyser et prédire le comportement des clients d'une entreprise de télécommunications afin d'identifier ceux qui risquent de résilier leur abonnement (Churn). L'objectif est de permettre à l'entreprise de prendre des mesures proactives pour améliorer la rétention.
